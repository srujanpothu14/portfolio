# Portfolio

Personal portfolio built with Angular 19 and SCSS.

## Tech Stack

- Angular 19 (standalone components)
- TypeScript (strict mode)
- SCSS

## Project Structure

```text
src/
	app/
		intro-page/
		about-page/
		experience-page/
		projects-page/
		skills-page/
		education-page/
		certifications-page/
		contact-page/
		app.component.*
	assets/
	styles.scss
```

## Available Scripts

```bash
npm start      # run dev server (ng serve)
npm run build  # production build
npm test       # run unit tests
```

## Local Development

1. Install dependencies:

```bash
npm install
```

2. Start the app:

```bash
npm start
```

3. Open:

`http://localhost:4200`

## Notes

- Production build output is generated in `dist/portfolio`.
- This repository uses strict TypeScript and Angular template checks.
